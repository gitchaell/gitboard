# **App Name**: GitBoard

## Core Features:

- Data Fetching: Fetch GitHub data, focusing on commits, contributions, pull requests, and activity per user, using GitHub's GraphQL API.
- Global Ranking: Create a ranking of top GitHub users by country based on aggregated statistics, updated in real-time.
- Detailed User Profiles: Allow users to search for and view individual user profiles, including a detailed breakdown of their contributions, languages used, and recent activity.
- Language Statistics: Provide aggregated data on the most used programming languages globally and per country.
- UI Components Integration: Use Zard.UI library for consistent, pre-styled UI elements and components throughout the dashboard.

## Style Guidelines:

- Primary color: Dark slate blue (#455A64) to provide a professional and analytical feel.
- Background color: Light grey (#EEEEEE) to ensure readability and focus on the data.
- Accent color: Teal (#009688) for interactive elements and highlights, contrasting effectively with the background.
- Body and headline font: 'Geist' sans-serif font for a modern and neutral aesthetic and Geist Mono for numbers, dates, codes, etc.
- Use flat, geometric icons to represent different data types and actions. Icons should align with Zard.UI's visual style. Do not use emojis.
- Implement a grid-based layout to organize data visualizations, rankings, and user profiles. Prioritize readability and data presentation.
- Use subtle transitions and animations for data updates and user interactions. Animations should enhance the user experience without being distracting. View Transitions.
